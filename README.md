# نظام الحضور والغياب - خبراء العطور الدولية

نظام شامل لإدارة الحضور والغياب وتقييم الموظفين للشركات التجارية.

![IPE Logo](public/ipe-logo.png)

## المميزات

- ✅ نظام حضور وغياب بفترتين عمل (صباحية ومسائية)
- ✅ إدارة الأقسام والموظفين
- ✅ نظام تقييم شامل للموظفين
- ✅ ثلاثة مستويات صلاحيات (مدير، مشرف، موظف)
- ✅ تسجيل تلقائي للتأخير والغياب
- ✅ تقارير وإحصائيات مفصلة
- ✅ واجهة عربية كاملة مع دعم RTL
- ✅ تصميم عصري وسهل الاستخدام
- ✅ طرق متعددة لتسجيل الحضور (الموقع، البصمة، NFC، يدوي)
- ✅ تصدير التقارير إلى Excel و PDF

## التقنيات المستخدمة

- **Frontend**: Next.js 16, React 19, TypeScript
- **Styling**: Tailwind CSS v4, shadcn/ui
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Charts**: Recharts

## البدء

### المتطلبات

- Node.js 18+
- حساب Supabase

### التثبيت

1. استنساخ المشروع
2. تثبيت الحزم: `npm install`
3. إعداد متغيرات البيئة في Vercel
4. تشغيل سكريبتات قاعدة البيانات بالترتيب:
   - `001_create_database_schema.sql`
   - `002_create_rls_policies.sql`
   - `003_seed_initial_data.sql`
   - `004_seed_test_data.sql`

## الصلاحيات

### المدير (Manager)
- الوصول الكامل لجميع الميزات
- إدارة الأقسام والموظفين
- إدارة فترات العمل ومعايير التقييم
- عرض جميع التقارير والإحصائيات
- إعدادات النظام

### المشرف (Supervisor)
- عرض وإدارة موظفي قسمه
- تسجيل حضور الموظفين
- إجراء تقييمات الموظفين
- عرض تقارير قسمه

### الموظف (Employee)
- تسجيل الحضور والانصراف
- عرض سجل حضوره
- عرض تقييماته
